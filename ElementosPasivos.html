Elementos Pasivos

Aprende más acerca de los elementos pasivos.

RJ45

El RJ45 es un conector estándar para cables Ethernet, utilizado en dispositivos de red como routers y módems. Facilita la transmisión de datos a través de cables de par trenzado.




Elementos Pasivos
Jack RJ45 o Keystone

Un conector keystone es un módulo hembra usado en redes LAN, montado en paneles de pared o conexión. Su versatilidad permite instalar múltiples jacks keystone en un panel y conectar diferentes tipos de cables y conductores. Los conectores keystone blindados protegen los datos de interferencias electromagnéticas.

Elementos Pasivos
Cable UTP

El cable UTP es un cable de red económico con pares trenzados y una cubierta plástica, sin blindaje. Usado para LAN Ethernet, es flexible y fácil de instalar, con cables Cat 5e y Cat 6 adecuados para Gigabit Ethernet.

Elementos Pasivos
Path panel

Un patch panel es un dispositivo con múltiples puertos para gestionar cables LAN, fibra o cobre.

      Fibra óptica: Estándar (adaptadores LC/SC/MTP), breakout (alta densidad), modular (personalizable).
      Ethernet: Cat5e/Cat6 (cobre, paso o punzonado) y Keystone en blanco (personalizable, montaje en rack).

Elementos Pasivos
Patch Cord

Un patch cord es un cable utilizado para conectar dispositivos electrónicos, como ordenadores y routers, generalmente con conectores RJ-45. Estos cables, hechos de cobre y cubiertos de plástico, se usan en redes y telecomunicaciones.

Elementos Pasivos
Fibra Óptica

La fibra óptica transmite datos a alta velocidad mediante pulsos de luz en un filamento delgado de vidrio o plástico. Usa la Ley de Snell para mantener la luz dentro del núcleo. Tipos:

      Monomodal: Para largas distancias y alta velocidad.
      Multimodal: Para distancias cortas, con mayor margen.

Elementos Pasivos
Canaletas

Una canaleta para cables es una estructura de plástico o metal con tapa, usada para organizar y proteger cables eléctricos. Ofrece resistencia a la corrosión, fácil instalación, versatilidad y durabilidad.
